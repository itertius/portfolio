Research & Internship

Workshop Paper · CVPR 2026
UColor: Advanced Seasonal Color Analysis for Perceptually-Aware Image Classification
CVPR 2026 · FGVC Workshop Submission

UColor addresses a fundamental gap in Computer Vision: current models perceive color in RGB without understanding human color perception. The project proposes a two-stage architecture using FaRL backbone with a hybrid perceptual loss (Cross-Entropy + CIEDE2000 ΔE) to classify Seasonal Colors (Spring, Summer, Autumn, Winter) in a perceptually-aware manner — outperforming Stacchio et al. at fewer training epochs.

Research directions: Hierarchical Classification (Warm/Cool → Seasonal), Perceptual-Aware Learning with CIELAB color space, Color Branch Network, and Advanced Evaluation via ΔE distribution and class separability.
