Conference Paper · CVPR 2026

UColor: Advanced Seasonal Color Analysis for Perceptually-Aware Image Classification

CVPR 2026 · Full Paper

UColor addresses a fundamental gap in Computer Vision: current models perceive color in RGB without understanding human color perception. The project proposes a two-stage architecture using FaRL backbone with a hybrid perceptual loss (Cross-Entropy + CIEDE2000) to classify Seasonal Colors in a perceptually-aware manner.
