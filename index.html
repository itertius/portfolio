Research & Publications

Workshop Paper · FGVC2026

UColor: Advanced Seasonal Color Analysis for Perceptually-Aware Image Classification

FGVC2026 Workshop · Non-archival 4-pages submission

UColor addresses a fundamental gap in Computer Vision: current models perceive color in RGB without understanding human color perception. The project proposes a two-stage architecture using FaRL backbone with a hybrid perceptual loss (Cross-Entropy + CIEDE2000) to classify Seasonal Colors in a perceptually-aware manner.

Research directions: Hierarchical Classification, Perceptual-Aware Learning with CIELAB color space, Color Branch Network, and Advanced Evaluation via distribution analysis.

Computer Vision Color Science FaRL CIEDE2000 CIELAB PyTorch Perceptual Loss
Internship · Dec 2025 - Feb 2026

Uphasia Artificial Intelligence Co., Ltd.

Researched and developed Computer Vision and Color Analysis models. Applied Color Science principles: CIELAB color space and CIEDE2000 metric. Developed Deep Learning models for Seasonal Color classification and designed perceptual-aware loss functions.
